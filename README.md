# gov.il (data-gov-il)

gov.il is a open data portal open-data portal for isr running **CKAN**, exposing a standardized catalog API over ~1,194 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-gov-il/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-gov-il-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN

## API
- **gov.il CKAN Action API** — base `https://data.gov.il/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.gov.il)
- Live endpoint verified: `https://data.gov.il/api/3/action/package_search?rows=0` · ~1,194 datasets
